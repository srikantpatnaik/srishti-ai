# Rust Tools

Optimized Rust-based tools for file operations, bash execution, and project management.

## Features

- **FileOps**: Read, write, delete, list files with minimal overhead
- **BashOps**: Execute commands with timeout support
- **CodeOps**: Format, lint, test, and build code
- **ProjectOps**: Scaffold Next.js and React projects

## Building

```bash
cargo build --release
```

## Testing

```bash
cargo test
```

## Usage in Next.js

The Rust tools are designed to be called from the Next.js API routes for maximum efficiency.
