Srishti AI - AI-powered app builder PWA

What it does:
- User types in chat interface
- AI autonomously builds web apps (HTML/CSS/JS)
- Generates images and audio
- Live preview with dev console
- Gallery to save/share apps

Intent Routing:
1. User request → intent-detector.ts (regex + keyword matching)
2. Image/audio intent → /api/router → SD API / TTS API
3. Text intent → /api/chat → Ollama (qwen3.6-35B)
4. Frontend renders HTML preview

Tech Stack:
- Next.js 14 + App Router
- Vercel AI SDK
- Ollama (qwen3.6-35B on LAN)
- Stable Diffusion API for images
- TTS API for audio
- Tailwind CSS + shadcn/ui
- next-pwa for offline support

Key files:
- app/api/chat/route.ts - Text streaming API
- app/api/router/route.ts - Intent routing (image/audio)
- lib/intent-detector.ts - Intent detection patterns
- lib/clients.ts - Ollama client
- app/page.tsx - Main UI
- settings.yaml - Provider config
