Srishti AI - AI-powered app builder PWA

What it does:
- User types in chat interface
- AI autonomously builds web apps (HTML/CSS/JS)
- Live preview with dev console
- Error detection + auto-fix
- PWA installable on mobile

AI Pipeline:
1. User request → /api/chat
2. Plano gateway (localhost:12000) routes to:
   - gpt-4o for code generation
   - gpt-4o-mini for simple QA
   - qwen3.5-4B (llama.cpp) for local/offline tasks
3. Model generates code
4. Frontend renders HTML preview

Tech Stack:
- Next.js 15 + App Router
- Vercel AI SDK (Google Gemini + OpenAI)
- Tailwind CSS + shadcn/ui
- Rust tools for file ops
- next-pwa for offline support

Key files:
- app/api/chat/route.ts - AI chat API
- app/page.tsx - Main UI
- plano_config.yaml - Routing config
- rust-tools/ - Optimized Rust utilities
